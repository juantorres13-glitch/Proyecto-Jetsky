## Descripción del Proyecto
Este proyecto consiste en el desarrollo de un sistema web para la gestión de reservas y venta de servicios de una aerolínea.

El sistema permitirá:

- Registro y gestión de clientes
- Reserva de vuelos
- Compra de tiquetes
- Adquisición de paquetes turísticos
- Administración de reservas

## Tecnologías a utilizar
- HTML5
- CSS (Bootstrap / Flexbox)
- JavaScript
- PostgreSQL
