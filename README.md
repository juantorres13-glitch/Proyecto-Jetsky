## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un sistema web para la **gestión de reservas y venta de servicios de una aerolínea**.

El sistema permitirá:

* Registro y gestión de clientes
* Reserva de vuelos nacionales e internacionales
* Compra de tiquetes aéreos
* Adquisición de paquetes turísticos adicionales
* Administración del estado de reservas

---
